# PysicsSimulator

**PysicsSimulator** is a web-based physics simulation project developed by **Wissam Ismail**. The repository contains a collection of interactive simulators designed to demonstrate various physical principles through a browser-based interface.

## Features

*   **Multiple Simulation Modules:** The project includes seven distinct simulation environments, labeled **Simulator1 through Simulator7**.
*   **Web-Based Interface:** Each simulator has a corresponding HTML entry point (e.g., `simulator1.html`, `simulator2.html`), allowing them to be run directly in a web browser.
*   **Modular Design:** Features a `Common` directory, which likely houses shared assets, styles, or logic used across all simulation modules.
*   **VS Code Integration:** Includes workspace configurations (`wsSimulator.code-workspace`) for easy development and management within Visual Studio Code.

## Technical Stack

*   **JavaScript (57.8%):** Drives the core logic and mathematical calculations for the physics simulations.
*   **HTML (21.5%):** Provides the structure for the simulator interfaces.
*   **CSS (20.7%):** Handles the visual presentation and layout of the simulation environments.

## Repository Structure

The repository is organized into specific folders for each simulator and its supporting files:
*   **`Simulator1/` to `Simulator7/`**: Folders containing the specific logic and assets for each individual simulation.
*   **`simulator1.html` to `simulator7.html`**: The main HTML files used to launch each simulator.
*   **`Common/`**: Shared resources and code utilized by multiple simulators.
*   **`.vscode/` & `wsSimulator.code-workspace`**: Configuration files for the Visual Studio Code development environment.

## Getting Started

### Prerequisites
*   A modern web browser (Chrome, Firefox, Safari, or Edge).

### Running the Simulators
1.  **Clone the repository** to your local machine.
2.  Navigate to the root directory.
3.  Open any of the simulator HTML files (e.g., **`simulator1.html`**) in your web browser to start the specific simulation.
